# Monthly Performance V3.0.1

เวอร์ชันแก้ไขการเชื่อมต่อ Google Apps Script อัตโนมัติ

## สิ่งที่แก้ไข

- ฝัง Web App URL เริ่มต้นไว้ใน `app.js`
- เปิดเว็บแล้วเชื่อมต่อ Backend และโหลดข้อมูลทันที
- เก็บ URL สำรองใน `localStorage`
- ลองเชื่อมต่อซ้ำอัตโนมัติเมื่อ Apps Script ตอบช้า
- ไม่เข้าสู่โหมดทดลองเอง เว้นแต่ผู้ใช้กดปุ่มโหมดทดลอง
- เมื่อบันทึก URL ใหม่ ระบบจะออกจากโหมดทดลองอัตโนมัติ

## ไฟล์ที่ต้องอัปเดตบน GitHub

แทนที่ไฟล์ `app.js` เป็นอย่างน้อย หรือแทนที่ `index.html`, `styles.css`, `app.js` ทั้งชุด

ไม่ต้องเปลี่ยน `Code.gs` และไม่ต้อง Deploy Google Apps Script ใหม่
